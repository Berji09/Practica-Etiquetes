Etiquetes

Etiqueta "cite"

Luis dijo:"... y al final aprobó todos los módulos".

Esta etiqueta marca una referencia a una fuente, o el autor de un texto citado.

Etiqueta "q"

Personalmente no creo que ...de lo contrario tendré que ir a recuperaciones.

Esta etiqueta permite a los autores insertar citas en forma de líneas cortas de texto.

Etiqueta "dfn"

El HTML es un lenguaje de marcado para hipertextos.

Esta etiqueta sirve para marcar el término que se quiere definir.

Etiqueta "abbr"

El Sr.Iván Berjillos estudió en el Ies el Grao.

Esta etiqueta representa a un término abreviado o acrónimo.
